# servowatch
A simple project about monitoring the services


## Project Setup

### Version
python 3.10+

### Create a virtual environment

Create,

```
python3.10 -m venv penv
```

Activate,
```
. penv/bin/activate
```

### Install python packages

```
pip install -r requirements.txt
```

### Run the script

```
python -m monitor_service
```

### Run the flask application
```
python -m app
```
